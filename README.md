
# Bank Loan Analysis using Python

## Project Overview
This project analyzes bank loan data to understand patterns in loan approval and customer financial behavior. The goal is to identify key factors that influence loan approval decisions.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Steps
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Insights on Loan Approval

## Dataset Features
- Loan Amount
- Applicant Income
- Credit History
- Education
- Property Area
- Loan Status

## Key Insights
- Credit history plays a major role in loan approval
- Higher income applicants have higher approval chances
- Property area influences loan approval rates
